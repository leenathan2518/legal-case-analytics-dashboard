# Notary Case Analytics Dashboard

## Live Demo

https://legal-case-analytics-dashboard-gkqbborzomstuaxxbikwxy.streamlit.app/

Interactive analytics dashboard for notary case management built with MySQL, Python, Streamlit, Pandas, and Plotly.

---

## Project Overview

This project simulates a real-world legal service analytics pipeline using anonymized notary case records.

The workflow includes:

- Raw data ingestion
- SQL-based ETL pipeline
- Data cleaning and validation
- Relational database normalization
- Staff desensitization and privacy protection
- Reporting view generation
- CSV export pipeline
- Interactive Streamlit dashboard

The dashboard provides business insights into:

- Case volume trends
- Fee revenue analysis
- Case type distribution
- Notary performance metrics
- Interactive filtering and downloadable reports

## Data Privacy

This public portfolio version uses anonymized datasets.

Sensitive information such as:
- Client names
- Phone numbers
- Identification numbers
- Staff names

has been removed or replaced with anonymized identifiers.

---

## SQL Pipeline

The SQL workflow includes:

1. Database setup
2. Data profiling and EDA
3. Table creation
4. Data cleaning
5. Date normalization
6. Multi-party parsing
7. Staff mapping and desensitization
8. Aggregated metrics generation
9. Reporting views
10. Dashboard dataset export
